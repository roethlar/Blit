# GEMINI.md

`AGENTS.md` at the repo root is the canonical agent contract — Antigravity CLI
loads it natively alongside this file. If your harness loaded only GEMINI.md,
read `AGENTS.md` now and follow it, starting with `docs/STATE.md`.

Workspace skills in `.agents/skills/` mirror part of the trigger vocabulary
(`catchup`, `handoff`); the full procedure set is `docs/agent/PROTOCOL.md`.
